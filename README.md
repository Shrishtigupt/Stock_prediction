# 📈 Stock Price Prediction App

## Overview

This project is a **Stock Price Prediction Web Application** built using **Python, Streamlit, Prophet, Pandas, Matplotlib, and Yahoo Finance API**.

The application allows users to:

* Select stocks from a dropdown list
* View historical stock market data
* Predict future stock prices
* Generate long-term forecasts up to the year 2027
* Visualize historical and predicted trends using graphs

The project is deployed using **Streamlit Community Cloud** and connected with GitHub.

---

#  Features

## ✅ Historical Stock Data

* Fetches stock market data using Yahoo Finance API
* Displays historical closing prices
* Supports both Indian and international stocks

## ✅ Future Stock Prediction

* Uses Prophet forecasting model
* Predicts future stock prices for multiple years
* Generates predictions up to the year 2027

## ✅ Interactive User Interface

* Built using Streamlit
* Dropdown stock selector
* Clean and user-friendly dashboard

## ✅ Data Visualization

* Historical stock price graph
* Forecast trend graph
* Future predicted values table

# ⚙️ How It Works

1. User selects a stock from the dropdown menu.
2. The app downloads stock data from Yahoo Finance.
3. Data is cleaned and processed.
4. Prophet forecasting model is trained on historical stock data.
5. Future stock predictions are generated.
6. Results are displayed using tables and graphs.
